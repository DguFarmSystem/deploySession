# deploySession
보안/웹 배포 자동화 학습 세션

1. main -> resources -> application.yml 생성 (빈 파일)
  - 아래 내용 그대로 첨부
```
server:
  port: 8080
```
2. main -> java -> farm.deploySession -> DeploySessionApplication에서 main을 ▶️RUN(초록색 화살표)
   - 웹 브라우저에서 `http://localhost:8080/`로 접속 후 아래 화면 뜨면 성공
